# Examples

Runnable scripts live in `examples/`.

- `examples/refinements.py` shows `refined`, `where`, and known predicates.
- `examples/checked_functions.py` shows argument validation, return validation, and `ensure`.
- `examples/sized_collections.py` shows shared symbolic lengths.
- `examples/numpy_shapes.py` shows optional NumPy shape and dtype validation.
- `examples/proofs.py` shows `Proof`, `prove`, and `RequiresProof`.
- `examples/registries.py` shows registry metadata.

The topic pages in this site expand each of those patterns.
